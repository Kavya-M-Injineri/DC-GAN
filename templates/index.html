About This Application

This application uses a Deep Convolutional Generative Adversarial Network (DC GAN) trained on the MNIST dataset to generate synthetic handwritten digit images. The generator network takes random noise vectors as input and produces 28x28 grayscale images that resemble handwritten digits (0-9).

API Endpoints

POST /api/generate - Generate multiple images (body: {"num_images": N}) POST /api/generate-single - Generate a single image GET /api/status - Check model status GET /api/health - Health check
